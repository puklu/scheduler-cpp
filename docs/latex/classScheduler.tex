\doxysubsection{Detailed Description}
Manages resource allocation and task scheduling. 

The \mbox{\hyperlink{classScheduler}{Scheduler}} class is responsible for managing resource allocation and task scheduling. It maintains a list of resources and project queues, allowing tasks to be added and allocated to resources based on their priority.
